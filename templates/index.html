📚 ТЕОРЕТИЧЕСКИЙ СПРАВОЧНИК

🔢 НАТУРАЛЬНЫЕ ЧИСЛА

📌 Определение:

Натуральные числа — это числа, которые используются для счета предметов: 1, 2, 3, 4, 5, ...

📌 Свойства:

  • Наименьшее натуральное число — 1
  • Наибольшего натурального числа не существует
  • Каждое натуральное число можно разложить на простые множители

🔢 ПРОСТЫЕ И СОСТАВНЫЕ ЧИСЛА

📌 Простые числа:

Это числа, которые имеют только два делителя: 1 и само себя.

Примеры: 2, 3, 5, 7, 11, 13, 17, 19, 23...

📌 Составные числа:

Это числа, которые имеют больше двух делителей.

Примеры: 4, 6, 8, 9, 10, 12, 14, 15, 16...

📌 Важно:

Число 1 не является ни простым, ни составным.

🔢 ДЕЛИМОСТЬ ЧИСЕЛ

📌 Делитель числа:

Делитель — это число, на которое данное число делится без остатка.

📌 Кратное числа:

Кратное — это число, которое делится на данное число без остатка.

📌 Признаки делимости:

  • На 2: число четное
  • На 3: сумма цифр делится на 3
  • На 5: число оканчивается на 0 или 5
  • На 9: сумма цифр делится на 9
  • На 10: число оканчивается на 0

🔢 НОД И НОК

📌 НОД:

Наибольший общий делитель — это самое большое число, на которое делятся все данные числа.

Обозначение: НОД(a, b)

📌 НОК:

Наименьшее общее кратное — это самое маленькое число, которое делится на все данные числа.

Обозначение: НОК(a, b)

📌 Свойство:

НОД(a, b) × НОК(a, b) = a × b

🥧 ОБЫКНОВЕННЫЕ ДРОБИ

📌 Структура дроби:

a/b

  • a — числитель (сколько взяли)
  • b — знаменатель (на сколько частей разделили)

📌 Правильная дробь:

Числитель меньше знаменателя (a < b)

📌 Неправильная дробь:

Числитель больше или равен знаменателю (a ≥ b)

📌 Смешанное число:

Целая часть и дробная часть

🔟 ДЕСЯТИЧНЫЕ ДРОБИ

📌 Определение:

Десятичная дробь — это дробь со знаменателем 10, 100, 1000 и т.д.

📌 Разряды:

  • Первая цифра после запятой — десятые
  • Вторая цифра после запятой — сотые
  • Третья цифра после запятой — тысячные

➖ ОТРИЦАТЕЛЬНЫЕ ЧИСЛА

📌 Определение:

Отрицательные числа — это числа меньше нуля. Они обозначаются знаком минус.

📌 Модуль числа |a|:

Модуль — это расстояние от числа до нуля на координатной прямой. Модуль всегда положительный.

📌 Противоположные числа:

Числа, которые отличаются только знаком: a и −a

⚖️ ПРОПОРЦИИ

📌 Определение:

Пропорция — это равенство двух отношений.

a : b = c : d

📌 Крайние и средние члены:

  • a и d — крайние члены
  • b и c — средние члены

📌 Основное свойство:

Произведение крайних членов равно произведению средних

a × d = b × c

💯 ПРОЦЕНТЫ

📌 Определение:

Процент — это сотая часть числа.

1% = 1/100 = 0.01

📌 Перевод:

  • Чтобы перевести проценты в дробь — разделить на 100
  • Чтобы перевести дробь в проценты — умножить на 100

🗺️ МАСШТАБ

📌 Определение:

Масштаб — это отношение длины отрезка на карте к длине соответствующего отрезка на местности.

📌 Виды масштаба:

  • Численный (1 : 10000)
  • Именованный (в 1 см — 100 м)
  • Линейный (графический)

📈 КООРДИНАТНАЯ ПЛОСКОСТЬ

📌 Элементы:

  • Ось X (абсцисс) — горизонтальная
  • Ось Y (ординат) — вертикальная
  • Начало координат — точка пересечения осей (0;0)

📌 Координаты точки:

A(x; y)

где x — координата по оси X, y — координата по оси Y

📌 Четверти:

  • I четверть: x > 0, y > 0
  • II четверть: x < 0, y > 0
  • III четверть: x < 0, y < 0
  • IV четверть: x > 0, y < 0

📐 ГЕОМЕТРИЧЕСКИЕ ФИГУРЫ

📌 Окружность и круг:

  • Радиус (R) — отрезок от центра до точки на окружности
  • Диаметр (D) — отрезок, соединяющий две точки окружности через центр (D = 2R)
  • Длина окружности: C = 2πR
  • Площадь круга: S = πR²

📌 Прямоугольник:

  • Площадь: S = a × b
  • Периметр: P = 2(a + b)

📌 Квадрат:

  • Площадь: S = a²
  • Периметр: P = 4a

🌀 СИММЕТРИЯ

📌 Осевая симметрия:

Фигура симметрична относительно прямой (оси симметрии), если для каждой точки фигуры симметричная ей точка также принадлежит фигуре.

📌 Центральная симметрия:

Фигура симметрична относительно точки (центра симметрии), если для каждой точки фигуры симметричная ей точка также принадлежит фигуре.

✖️ УРАВНЕНИЯ

📌 Определение:

Уравнение — это равенство, содержащее неизвестное число (переменную).

📌 Корень уравнения:

Корень уравнения — это значение переменной, при котором уравнение становится верным равенством.

📌 Правила решения:

  • Перенос слагаемых с противоположным знаком
  • Деление обеих частей на одно и то же число
  • Умножение обеих частей на одно и то же число

✖️ свойства уровнении

📌 Определение:

📘 Справочник Свойства уравнений 🔹 1. Что такое уравнение Уравнение — это равенство, содержащее переменную, значение которой нужно найти. Значение переменной, при котором равенство становится верным, называется корнем уравнения. ⸻ 🔹 2. Основные свойства уравнений Эти свойства позволяют преобразовывать уравнение, не изменяя его корней. 1️⃣ Свойство сложения (вычитания) Если к обеим частям уравнения прибавить или из обеих частей вычесть одно и то же число (или выражение), равенство не изменится. 📌 Используется при переносе слагаемых из одной части уравнения в другую. ⸻ 2️⃣ Свойство умножения Если обе части уравнения умножить на одно и то же число, не равное нулю, равенство сохранится. ⸻ 3️⃣ Свойство деления Если обе части уравнения разделить на одно и то же число, не равное нулю, равенство сохранится. 📌 Используется для нахождения переменной, когда она имеет коэффициент. ⸻ 🔹 3. Перенос слагаемых Перенос слагаемого из одной части уравнения в другую выполняется с изменением знака. Это основано на свойстве прибавления противоположного числа к обеим частям уравнения. ⸻ 🔹 4. Подобные слагаемые Слагаемые, содержащие одинаковую переменную в одинаковой степени, называются подобными. Их можно складывать или вычитать, приводя уравнение к более простому виду. ⸻ 🔹 5. Коэффициент Число, стоящее перед переменной, называется коэффициентом. Чтобы найти переменную, нужно разделить обе части уравнения на её коэффициент (если он не равен нулю). ⸻ 🔹 6. Равносильные уравнения Уравнения называются равносильными, если они имеют одинаковые корни. Все преобразования, основанные на свойствах уравнений, приводят к равносильным уравнениям. ⸻ 🔹 7. Алгоритм решения линейного уравнения 1. Раскрыть скобки (если есть). 2. Привести подобные слагаемые. 3. Перенести слагаемые с переменной в одну часть, числа — в другую. 4.
